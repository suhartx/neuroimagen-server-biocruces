# Manual De Usuario

1. Entrá en la GUI.
2. Confirmá que el estudio está anonimizado.
3. Seleccioná el fichero.
4. Pulsá `Enviar a procesamiento`.
5. Esperá a que el estado pase de `En cola` a `Procesando` y luego a `Completado`.
6. Descargá el PDF cuando aparezca el botón.

Si el estado es `Fallido`, avisá al administrador y no uses ese resultado.

Los informes son apoyo y requieren revisión clínica.
